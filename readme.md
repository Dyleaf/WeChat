#### 功能和特性
- 基于socket实现的c/s架构的的通信
- 服务器和客户心跳连接
- gson实现的消息通信机制
- 注册及登录
- 支持私聊和群聊。
- 动态更新用户列表以及用户消息提示
- 支持emoji表情,以及emoji表情选择器
- 服务器端数据库用户记录
